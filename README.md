# FOODHUB BACKEND SERVER

This server serves for project [FoodHub](https://github.com/iamvucms/foodhub)
## Installation

```bash
git clone https://github.com/iamvucms/foodhub-backend.git
cd foodhub-backend
yarn
node index.js 
```
